# 3D Interactive CV Receipt

An interactive, 3D CV rendered with WebGL. It utilizes custom particle physics to simulate a realistic cloth-like paper effect for floating receipts hanging from a wire.

## 🔗 Live Demo
**Check out the interactive CV here:** [luminiferousBOT.github.io/cv-receipt/](https://luminiferousBOT.github.io/cv-receipt/)

## Features
- **Custom Cloth Physics:** The receipts behave like physical paper swaying in the wind.
- **Interactive:** Click and drag the receipts to see the WebGL physics engine react in real time.
- **Procedural Textures:** The Profile, Projects, and Education data are dynamically drawn onto 3D textures using the Canvas API.
